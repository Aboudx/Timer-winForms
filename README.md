# ⏱️ Timer - WinForms C#

A simple **Timer Desktop Application** built with **C# WinForms**.

This educational project demonstrates how to build a countdown timer using **TimeSpan** for accurate time management. The application includes a live progress bar, percentage indicator, and a sound notification when the countdown finishes.

## ✨ Features

- ⏳ Countdown timer
- ▶️ Start, Pause, and Reset controls
- 🕒 Hours, Minutes, and Seconds input
- ⚡️ Accurate time management using **TimeSpan**
- 📊 Live progress bar with percentage
- 🔔 Sound notification when the timer finishes
- 🎨 Clean and user-friendly interface

## 🛠️ Technologies

- C#
- Windows Forms (WinForms)
- .NET

## 📚 Note

This is an **educational project** created to practice **C# WinForms**, **TimeSpan**, and desktop application development.
